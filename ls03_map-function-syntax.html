Syntax for specifying the function to map

Load packages

Load purrr and repurrrsive, which contains recursive list examples. listviewer provides the interactive list viewing widgets.

library(purrr)
## install_github("jennybc/repurrrsive")
library(repurrrsive)
library(listviewer)

Function specification

Flexible and concise syntax for specifying YOUR_FUNCTION

nms <- map_chr(got_chars, "name")
aliases <- map(got_chars, "aliases")
aliases <- set_names(aliases, nms)
map(aliases, function(x) paste(x, collapse = "|"))
